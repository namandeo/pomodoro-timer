# Pomodoro Timer

A clean, modern Pomodoro timer web application to boost your productivity.

## Features

- 25/5/15 minute Pomodoro technique timer
- Clean, modern UI with dark/light theme
- Customizable timer durations
- Session tracking
- Progress visualization
- Responsive design
